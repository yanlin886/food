# 言林餐饮收银系统

[言林餐饮连锁收银系统](https://yanlin.com/solution/store-cash) 面向餐饮连锁品牌。

该系统包含 4 个项目

- API 后端服务
- Administration 管理台
- 餐厅收银前台
- 微信小程序

系统支持 SaaS 租户模式。

# 产品

- 产品文档 https://doc.yanlin.com/
- 收银台演示 https://demo.yanlin.com/
- 管理台演示 https://food.yanlin.com/

# 技术栈

- 后端 Golang + Postgres
- 前端 React + AntDesign + Typescript
- 小程序端 Taro + React + Typescript

该代码库仅仅包含代码库样例

- `api` 订单 Golang API 测试样例
- `frontend` 收银台主页 React context + hook 样例
